## Results Used

The paper proves decidability of vector reachability, scalar reachability, and
point-to-point reachability by fractional linear transformations for finitely
generated semigroups in `SL₂(ℤ)`. Its algorithms translate matrix equations
into regular-language problems using canonical representatives of the modular
group.

For the mortality campaign, this closes the unimodular stratum of the
projective-incidence problem underlying `M₂(3)`.

## Scope and Hazards

`SL₂(ℤ)` is essential. The theorem does not decide scalar reachability for
arbitrary nonsingular integer `2 × 2` generators whose determinants may have
absolute value greater than one. Projective normalization moves that missing
case into matrix groups over larger arithmetic rings; it is not a formal
application of the paper.
